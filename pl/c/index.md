# C

Σχεδιάστηκε το 1972, τυποποιήθηκε αρχικά από το ANSI και στη συνέχεια από το ISO (ANSI C, C89, C99, C11, ...)

ISO Committee  WG14

## Primitive types

```
#include <stdint.h>

uint8_t, uint16_t, uint32_t, uint64_t
int8_t, int16_t, int32_t, int64_t
```

* static_assert
* overloading 
* atomics
* thread_local
